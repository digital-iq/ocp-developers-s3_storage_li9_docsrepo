Windows Packages: Prerequisites.

Prepare access, artifacts, storage, network, security, and disconnected sources before installing.

Prerequisites

Online Prerequisites

CheckCommand Or DecisionExpected Result
ServersPrepare 3 Windows Server nodes for production or 1 node for explicit single-node mode.All servers have stable DNS or IP addresses.
Artifact accessDownload the public MSI or ZIP release artifacts and checksums.Files match the published SHA256 checksums for the intended version.
Service accountRun PowerShell as an administrator or use an automation account with local admin rights.Account can create directories, write runtime state, and start Li9 S3 processes.
Storage pathsPersistent directories or attached disks exist under C:\ProgramData\Li9\S3Storage or the selected ZIP install path.Paths survive reboot and have enough capacity for data, metadata, WAL, and logs.
FirewallOpen S3, admin, health, and internal service ports.Clients and peer nodes can connect to ports 9000 through 9010 as required by the selected node role.
Endpoint securityTerminate external TLS at the load balancer or reverse proxy when exposing the gateway outside the trusted network.External clients use HTTPS while node-to-node test deployments can use the package default HTTP endpoints.
 
$PSVersionTable.PSVersion
Get-ComputerInfo | Select-Object OsName, OsVersion
Test-NetConnection 10.20.10.11 -Port 9000
Test-NetConnection 10.20.10.11 -Port 9001
Test-NetConnection 10.20.10.11 -Port 9008
Get-NetFirewallRule | Where-Object DisplayName -like "Li9 S3*"

Air-Gapped Prerequisites

Copy MSI, ZIP, and checksum artifacts from the public Windows package release into an internal package share or enterprise software distribution system. Verify checksums before installing.

$Li9NativeVersion = "1.1.0-alpha.14"
$Share = "\\fileserver\li9-s3\windows\v1.1.0-alpha.14"
$Msi = Join-Path $Share "li9-s3-storage-windows-$Li9NativeVersion.msi"
$Zip = Join-Path $Share "li9-s3-storage-windows-$Li9NativeVersion.zip"
$Checksums = Join-Path $Share "checksums.txt"
Get-ChildItem $Share
Get-FileHash $Msi -Algorithm SHA256
Get-FileHash $Zip -Algorithm SHA256
Get-Content $Checksums
Copy-Item $Msi C:\Temp\
Copy-Item $Zip C:\Temp\
