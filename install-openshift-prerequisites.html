Prerequisites

Prepare the cluster before creating OLM resources. The same runtime design applies to online and air-gapped installations; only the artifact source changes.

Installation Mode Conflict Check

Operator, Helm, Linux package, and Windows package deployments are separate lifecycle models. Before installing an operator-managed runtime into a namespace, remove any Helm release or host package deployment that manages the same Li9 S3 endpoint, PVCs, Services, CRs, or storage paths.
