Li9 S3 Storage Overview

Li9 S3 Storage runs on OpenShift, Linux, and Windows, providing a self-managed S3 endpoint with bucket provisioning, scoped credentials, policy enforcement, durable metadata, data protection, encryption, audit, and lifecycle automation.

Get started by role

Platform administrator

Install the operator, configure storage placement and failure domains, enable metrics, validate node maintenance, and expose bucket consumption workflows.

Install on OpenShift

Application engineer

Request a bucket, mount the generated Secret, configure AWS CLI or SDK endpoint overrides, and validate object read/write behavior from a pod.

Use a bucket

Service owner

Review data protection, feature boundaries, operational risk, performance expectations, and compatibility with S3 clients before onboarding workloads.
